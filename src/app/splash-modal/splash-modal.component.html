Welcome to New EPIC!

We have recently updated our Project Information Centre, you will notice a different look and feel.

To find a project:

  • Click on “Find Environmental Assessment Projects” on the front page or “Find Projects” in the header
  • Enter your project name in the search field on the mapping interface and select the map pin
  • A pop up with give you some details regarding the project selected, click “View Project Details” to go to the project landing page

Project Landing Pages:

Project information is now split over 3 tabs

  • Project Information
  • Commenting
  • Documents

Documents are now searchable, there is a search help located in the header to assist you.

If you have any questions please contact us at epicsupport.eao@gov.bc.ca.
